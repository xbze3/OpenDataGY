# OpenDataGY

_Building the open data ecosystem for Guyana._

---

A curated collection of **open datasets, developer projects, APIs, and digital resources related to Guyana**.

The goal of this project is to make it easier for **developers, researchers, and civic technologists** to build software and tools focused on Guyana.

## Categories

- **Datasets**
- **Open Source Projects**
- **GIS & Maps**
- **APIs**
- **Developer Resources**
